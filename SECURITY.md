# Security Policy

## Supported Versions

Normally only the latest major version is supported.

## Reporting a Vulnerability

If you believe you have discovered a security issue, please email [galaxy-committers@lists.galaxyproject.org](galaxy-committers@lists.galaxyproject.org). Please use `[SECURITY]` in the email title.
Alternatively you can report a security vulnerability using GitHub [private reporting](https://github.com/galaxyproject/bioblend/security/advisories/new).

We ask that you not disclose security issues publicly. We will provide you credit for the discovery when publicly disclosing an issue you reported.
